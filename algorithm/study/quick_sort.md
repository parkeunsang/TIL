# Quick sort

## 개요

- 평균적으로 가장 빠른 알고리즘(O(nlogn)), but 최악의 경우 시간 복잡도는 O(n^2)
- 불안정 정렬(같은 값일 경우 위치순서를 보장하지 못함)
- 분할 정복(divide and conquer)

**Process**

- Macro
  1. 피벗 설정
  2. 해당 피벗의 위치 정확히 찾기(피벗기준 왼쪽은 자기보다작은것, 오른쪽은 자기보다 큰것)
  3. 피벗 좌, 우 리스트를 대상으로 이를 반복(분할 정복) 후 결합
  4. 리스트의 크기가 1이하일때 중단
